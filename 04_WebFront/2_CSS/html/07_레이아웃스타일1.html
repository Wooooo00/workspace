화면 배치(표시) 방법 : display

      요소가 화면에 어떻게 보여질지 형식을 지정하는 속성
      - block : 화면을 수직 분할(행을 나눔, 한 줄 모두 차지)
      + width, height 지정 가능

      - inline : 화면을 수평 분할(하나의 행의 열을 나눔 (글자, 문자열이라고 생각))
                 + width, height 지정 불가능! 
      - inline-block : inline의 수평 분할 + block의 width, height 지정
      - none : 요소는 존재하나 화면에는 보이지 않게 한다. 
      - flex : 요소의 정렬 되는 방향, 요소 간의 간격을 유연하게 처리하는 형식
      - grid : 격자무늬 형식으로 화면을 쉽게 분할

   - 내부 영역 분할 요소

      -부모 요소 크기 : 너비 : 300px 높이 : 600px
      -상하 분할 비율 : 20 : 30 : 50 
      -배경색 자유

좌우 2분할

준비물
      [준비물]
      - 감싸는 요소
      - 내부 분할 요소
      - display : inline-block(가로 배치, 크기 지정0)
      - 가변 크기 단위(%)
      ** 주의 사항 ** 
      inline이라는 단어가 들어가면 글자처럼 취급된다
      (글자 사이에는 띄어쓰기가 있다!!)

display : none;

      요소는 존재하지만 화면에 표시되지 않음
      display : none; -> 화면에 보이지 않고, 영역도 공간도 차지하지 않는다.
      visibility : hidden; -> 화면에 보이지 않지만, 영역은 차지한다
