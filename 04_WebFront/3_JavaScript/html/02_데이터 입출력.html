innerText

      js에서 요소에 작성된 내용(content)를 읽거나 변경하는 속성

      - 내용만 읽어온다(내용에 태그가 포함 되어 있어도 읽어오지 않는다)
      - 변경되는 내용에 태그, 마크업 특수 문자가 포함 되어 있어도 태그는 해석 되지 않는다

innerhtml

      자바스크립트에서 요소 전체를(태그 + 내용) 읽어들이거나 변경하는 속성
      내용을 읽어올 때 태그 + 내용 모두를 포함해서 읽음
      내용을 변경할 대 태그, 마크업 특수문자가 포함

[window.] alert("내용") 

 
   브라우저에 팝업을 띄우는 함수

[window.]confirm("내용")

      -"내용"에 대한 확인/취소 결과를 얻고자 할 때 사용하는 함수로 
      -alert에 확인/취소 두 개의 버튼이 생성된다.
      확인 -> True가 반환된다
      취소 -> False가 반환된다

[window.]prompt("내용")

      - 텍스트를 입력할 수 있는 alert
        
      - 확인 -> 텍스트 입력 후 확인 클릭 시 입력한 텍스트가 그대로 반환
      - 취소 -> null 반환

위 버튼을 클릭하세요
